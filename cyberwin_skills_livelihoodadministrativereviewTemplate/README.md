项目介绍
本项目为 AI 专用民生行政救济文书生成技能，无前端页面、无后端服务、不联网运算。
专注解决大众维权最大痛点：
用户只会「民间口语叫法」，不知道官方标准政务术语、不知道对应受理部门。
技能内置 alias（民间叫法）→ stand（官方标准名）→ depart（受理部门）→ lawBase（法定依据） 全自动映射体系。
输入口语关键词，即可自动生成合规、可直接递交的：
行政复议申请书
行政履职监督申请书
核心能力
口语模糊匹配：拆迁、准生证、低保、摆摊罚款、社保、投诉维权等民间词汇自动识别
自动替换官方标准文书用语（符合行政机关接收规范）
自动匹配对应主管行政机关（被申请人）
自动挂载对应法律法规依据
零法律基础、标准化、零自定义错误格式
纯模板工具，合规开源，无法律咨询、无代理、无教唆
合规说明
国内合规
本技能仅提供通用标准化文书模板框架，属于政务格式辅助工具，不构成法律咨询、案件代理、诉讼指导。所有事实陈述、证据由用户自行据实填写，不涉及捏造、教唆、群体性维权风险。
出口合规（不违反出口管制法）
